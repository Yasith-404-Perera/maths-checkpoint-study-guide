# 📐 Cambridge Checkpoint Maths — Complete Study Guide

> A beautifully designed, interactive HTML study guide for the **Cambridge Lower Secondary Checkpoint Mathematics (0862)** examination, covering all four strands with formulas, worked examples, exam tips, and practice questions.

---

## ✨ Features

- **4 complete strands** — Number, Algebra, Geometry & Measure, Statistics & Probability
- **Strand switcher** — instantly switch between subjects with a single click; colour theme updates to match
- **Formula sheets** — every formula you need, clearly presented in dark-themed boxes
- **Worked examples** — step-by-step solutions with numbered steps for every major topic
- **Exam tips** — green tip boxes throughout with specific mark-earning advice
- **Common mistake warnings** — red warning boxes flagging the errors examiners penalise most
- **Practice questions** — exam-style questions at the end of each strand with reveal-on-click answers
- **Fully responsive** — works on mobile, tablet, and desktop
- **No dependencies** — pure HTML, CSS, and vanilla JavaScript; no frameworks, no internet required after loading

---

## 📚 Topics Covered

### Number
- Integers & directed numbers, BIDMAS
- Fractions (operations, mixed numbers)
- Decimals & percentages (multiplier method, reverse percentage)
- Powers, roots & index laws
- Ratio & proportion (direct, inverse, map scales)
- Primes, HCF & LCM (prime factorisation, Venn method)
- Standard form

### Algebra
- Expressions, expanding brackets (single & double), factorising
- Solving linear equations & equations with fractions
- Inequalities (solving, number lines, integer solutions)
- Sequences (arithmetic nth term, geometric, special sequences)
- Straight-line graphs (y = mx + c, gradient, parallel & perpendicular)
- Quadratic graphs
- Simultaneous equations (elimination & substitution)

### Geometry & Measure
- Angle rules (parallel lines, polygons, interior & exterior angles)
- Area & perimeter of 2D shapes (including circles, sectors, trapezium)
- Volume & surface area of 3D shapes (cylinders, cones, spheres, prisms)
- Transformations (translation, reflection, rotation, enlargement)
- Trigonometry (SOH CAH TOA, exact values)
- Pythagoras' theorem (Pythagorean triples)
- Constructions, loci & bearings

### Statistics & Probability
- Mean, median, mode, range, IQR
- Estimated mean from frequency tables
- Statistical diagrams (bar charts, pie charts, histograms, box plots, stem-and-leaf)
- Basic probability, complementary events
- Mutually exclusive & independent events
- Tree diagrams (with and without replacement)
- Sample space diagrams
- Scatter graphs & correlation (line of best fit, interpolation vs extrapolation)

## 🚀 How to Use

### Option 1 — Open locally
1. Download or clone this repository
2. Open `index.html` in any web browser
3. No server or internet connection required (fonts load from Google Fonts on first use)

### Option 2 — GitHub Pages (recommended)
1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your guide will be live at `https://yasith-404-perera.github.io/maths-checkpoint-study-guide/index.html`

### Option 3 — Direct link
If you've enabled GitHub Pages, share the direct link with classmates or teachers.

## 🖨️ Printing

The guide is print-friendly. Use **File → Print** (or Ctrl+P / Cmd+P) in your browser. The navigation bars hide automatically when printing. For best results:
- Set paper size to A4
- Enable "Background graphics" in print settings to preserve coloured formula boxes
- Print one strand at a time by switching to that strand before printing

## 🎨 Design

The guide uses:
- **Playfair Display** — serif display font for headings
- **JetBrains Mono** — monospace font for formulas and code labels

- Distinct colour themes per strand: amber (Number), rose (Algebra), teal (Geometry), violet (Statistics)
- Dark formula boxes for high visibility of mathematical expressions
- Green exam tip boxes and red warning boxes for quick visual scanning
